# Sales Data Analysis (Python)

## About this project

This is a simple data analysis project where I worked with a sales dataset (500 records).
The main goal was to understand the data and find useful insights like revenue, top products, and sales trends.

---

## What I did

* Loaded the CSV file using pandas
* Cleaned the data (handled missing values)
* Created a new column called Revenue
* Found total, average, max, and min revenue
* Identified top and least selling products
* Analyzed sales based on city and category
* Checked payment methods and ratings
* Created monthly sales trend
* Visualized data using charts

---

## Tools used

* Python
* Pandas
* Matplotlib

---

## Dataset

The dataset contains:
Date, Product, Category, Price, Quantity, City, Discount, Payment, Rating

---

## How to run

1. Install libraries:
   pip install pandas matplotlib

2. Run the Python file:
   python analysis.py

---

## Output

* Printed insights in console
* Charts like:

  * Revenue by product
  * City-wise sales
  * Monthly trend
